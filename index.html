    Ongoing Projects

    SMoL Tutor: an intelligent tutoring system for teaching the semantics of programming languages (i.e., how programming languages "work"). This system has been used at Brown and a few other institutions around the world. Please contact me or Shriram to get a copy of the system.

    Stacker: a notional machine (a graphical presentation of how computer programs execute). This notional machine is designed to help students form good mental models of scope, mutable variables, mutable data structures, and first-class functions. It is implemented as a Web App.
